Type of Project:

Final project for AP Biology

Languages Used:

Description

The program created from this project is designed to semi-automate the triage process. Although not fully functional and not up to the standard required to function properly in an actual emergency room environment, it provides a good model of our concept. The patient would answer questions about the reason for their visit and the severity of their symptoms and the program would sort the patient. The information about triage and how emergency rooms function was acquired through research and after understanding how it works, we developed a solution to the issue of overcrowding by allowing the user to input their own information and let the system sort and process their information, which included recommending which specialists to see.

Type of Project:

Final project for AP Computer Science

Languages Used:

Java

Description:

This project was a simulation of the FIRST Robotics Competition’s game, FIRST Stronghold. The game allows the player to control a robot and gain points by breaching defenses and shoot boulders into a tower. The probability of the player making the shot is dependent on how far away they are from the tower. At the end of the game, points from breached defenses and boulders shot at the tower, mimicking the scoring mechanics of the real-life game.
