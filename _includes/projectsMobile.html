Information Medical Database Collection (iMDC)

    Languages: Java

    Type of project: Final project for AP Biology

    Description

    This purpose of this project was to create a model of a solution for the issue of emergency room overcrowding. The model created from this was a program that would help to automate the triage process. The patient would answer questions about the reason for their visit and the severity of their symptoms. Based off of the responses, the program would sort and process their information and recommend which specialists to see.

Encryption Project

    Languages: Java

    Type of project: AP Computer Science project

    Description

    The task for this project was to create a program that could encrypt and decrypt messages with a method of encryption of our choice. This program encrypts messages with a substitution cipher. The user inputs an alphabet, a word, and the message they want to encrypt. If given the alphabet, word, and an encrypted message, the program is able to decrypt said message.
